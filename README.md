# OctoStore-Releases
OctoStore is a Fork of 8Store, a discontinued Windows 8 Store Revival.
OctoStore is based on 8Store version 5.3 and aims to replicate 8Store but also make a few things different in it's own way.
This repo here is here to archive all the versions of OctoStore so it won't be lost media.
This revival is made by Slight Commercial if you are wondering and not me.
